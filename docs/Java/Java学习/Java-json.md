# Java-json

- json基础

  - 特性

    - 只允许用UTF-8编码

    - 只允许使用双引号作为key，如"name"，特殊字符用反斜杠转义

    - 只支持以下几种数据类型：

      - 键值对：{"key": value}

      - 数组：[1, 2, 3]

      - 字符串："abc"

      - 数值（整数和浮点数）：12.34

      - 布尔值：true或false

      - 空值：null
