# Java对象访问限制

- 类修饰符

  - public

    - 可被所有其他类访问

  - 不指定修饰符

    - 仅可被同一包中其他类访问（package-private)

- 成员修饰符

  - public或不指定修饰符

    - 与类情况相同

  - private

    - 仅可被同一类的其他成员访问

  - protect

    - 可被同一包中其他类访问

    - 或其他包中的子类（子类仅可访问自己的成员，不可访问超类的成员，因为不在同一个包中）

      - Java官网只说子类能访问超类protected成员，显然是不完整的
