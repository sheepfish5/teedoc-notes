解释闭包

2024-02-16

参考：MDN - 闭包及其应用

function makeAdder(x) {
  return function (y) {
    return x + y;
  };
}

var add5 = makeAdder(5);
var add10 = makeAdder(10);

console.log(add5(2)); // 7
console.log(add10(2)); // 12

上面的代码中，add5和add10都有着各自的私有作用域，里面有变量x的定义，像这样保存了不同的词法环境的函数，以及声明该函数的词法环境的组合，就叫做闭包。补充一点，有时不一定是函数，比如javascript对象加上其独特的作用域，也算是闭包。
