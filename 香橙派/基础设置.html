基础设置

2024-02-16

1. 设置ssh连接

跟着阮一峰老师的教程走就行了。

主要设置了以下几点：

  1. 上传本地公钥到香橙派上，实现免密码登陆(注意上传公钥可以直接使用ssh-copy-id命令)
  2. 修改本地配置文件，使用别名连接香橙派

2. 安装docker

注意不要直接安装手册上推荐的包，那些不是arm版上的，而应该安装docker.io和docker-compose。

安装好两个包后，还要设置国内阿里云的image源，我已经用电信流量卡的手机号注册过了，直接登陆阿里云就可以看到账号的专属源地址了。

在香橙派上初次运行docker run --rm hello-world会报AppArmor的错，根据StackOverflow上的回答，是系统没装AppArmor，装上后再重启docker就好了：apt install apparmor -y

3. 同步时间

安装ntpdate，修改时区，设置crontab任务设置时间同步：

sudo apt install ntpdate -y

sudo dpkg-reconfigure tzdata
# 出现界面，先选择选择Asia
# 然后再选择Shanghai

sudo crontab -e
# 加上：*/60 * * * * /usr/sbin/ntpdate ntp.sjtu.edu.cn
